An interactive 3D visualization tool for wellbore schematics built with Python. This project demonstrates the application of **Digital Transformation** in the Oil & Gas industry by converting static well design data into intuitive, interactive visual models.

## 🌟 Key Features

- **270° Cutaway View:** Implements trigonometric meshgrid calculations to generate a sliced view, allowing for detailed inspection of inner casing and cement layers.
- **Data-Driven Architecture:** All well parameters (depth, radii, components, and colors) are managed via a central `well_data.csv` file. Visualize new wells instantly without touching the source code.
- **Comprehensive Component Mapping:** Supports multiple casing strings including Conductor, Surface, Intermediate, Production Casing, and Liners, along with their respective cement sheaths.
- **Full Interactivity:** Powered by Plotly, offering real-time rotation, zooming, and hover-sensitive tooltips for precise data analysis.

- **Python**: Core programming language.
- **Plotly**: For high-performance interactive 3D rendering.
- **Pandas**: Efficient management and manipulation of tabular `.csv` data.
- **NumPy**: Numerical computing for meshgrid coordinate generation.
